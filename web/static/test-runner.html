Key Management

                    

                        key generate
                        Generate a new keypair (nsec + npub)
                    

                    

                        key public
                        Derive public key from private key (requires stdin)
                    

                    

                        key encrypt
                        Encrypt private key with password (NIP-49)
                    

                    

                        key decrypt
                        Decrypt encrypted private key (NIP-49)
                    

                

                

                    
Encoding/Decoding

                    

                        decode <bech32>
                        Decode npub, nsec, note, nevent, nprofile, naddr
                    

                    

                        encode npub <hex>
                        Encode hex pubkey to npub format
                    

                    

                        encode nsec <hex>
                        Encode hex privkey to nsec format
                    

                

                

                    
Event Operations

                    

                        event --content <text>
                        Create unsigned kind 1 text note
                    

                    

                        verify
                        Verify event signature (requires stdin)
                    

                

                

                    
Relay Queries

                    

                        req -k <kind> -l <limit>
                        Query events by kind with limit
                    

                    

                        req -a <pubkey>
                        Query events by author pubkey
                    

                

                

                    
Fetch & Publish

                    

                        fetch <nevent|nprofile|naddr>
                        Fetch event by bech32 identifier
                    

                

                

                    
NIP-05 & DNS

                    

                        nip05 <identifier>
                        Resolve NIP-05 identifier to pubkey
                    

                

                

                    
Utility

                    

                        --help
                        Show all available commands
                    

                    

                        --version
                        Show nak version
